destroy

vt 重难点
英音 [dɪ'strɔɪ]
美音 [dɪ'strɔɪ]
词频 4口语 2书面 2

英文释义

  • ★ 毁坏；摧毁；毁掉 (to damage sth so badly that it can no longer be used or no longer exists：)

    • The building was completely destroyed by fire.

      那座楼被火彻底烧毁了。

    • Everywhere they looked nearly everything was destroyed.

      他们所见之处几乎一切尽毁。

    • destroy sb’s confidence / hope / faith

      毁掉某人的信心/希望/信仰

  • 消灭；杀死：

    • The dog attacked a child and had to be destroyed.

      那只狗袭击了一个小孩，所以要被杀掉。
