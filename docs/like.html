like

vt 重难点
英音 [laɪk]
美音 [laɪk]
词频 5口语 3书面 3

用法点拨

[一般不用于进行时]

英文释义

  • ★ 喜欢：

    • What kind of movie do you like best?

      你最喜欢哪种类型的电影？

  • ★ 希望；想要：

    • I'd like a double room.

      我想要一个双人间。

    • Would you like some coffee or tea?

      你想喝点咖啡还是茶？

    • You can choose anything you like from the menu.

      菜单上的你可以随便点。

固定搭配

How do you like ...?

你 (们) 认为…怎么样？

  • How do you like this oil painting?

    你认为这幅油画怎么样？

I'd like to.

[省略句，作答语用] 我愿意。

    if you like

    如果你愿意：

    • I can take you there in my car, if you like.

      如果你愿意，我可以开车送你去那里。

        你一定要有所擅长，比如体育、新闻播报或表演。
