message

n
英音 ['mesɪdʒ]
美音 ['mesɪdʒ]
词频 5口语 3书面 2

用法点拨

[C]

英文释义

  • ★ 消息；口信：

    • take / leave a message

      传话/留言

  • ★ (电邮 / 手机) 信息：

    • send a text message

      发短信

  • ★ (书籍、演讲等的)要旨；主题思想：

    • The message of the film is that good always wins over evil.

      这部影片的要义是正义总能战胜邪恶。

固定搭配

get the message

领悟；理解；明白 (别人的暗示)：

  • — Mark, I think you’re the right person for the job.

    马克，我认为你是这个职位的合适人选。

  • — I get the message.

    我明白了。
