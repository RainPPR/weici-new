therefore

adv
英音 ['ðeəfɔː(r)]
美音 ['ðerfɔːr]
词频 5口语 2书面 3

英文释义

  • ★ 因此；所以(thus)：

    • I think, therefore I am.

      我思故我在。

    • We don’t need to do so much homework. Therefore, we have more time for after-school activities.

      我们不需要做太多作业，因此，有更多的时间参加课外活动。
