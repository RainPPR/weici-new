problem

n
英音 ['prɒbləm]
美音 ['prɑːbləm]
词频 5口语 3书面 3

用法点拨

[C]

英文释义

  • ★ 问题；难题：

    • health / family problems

      健康 / 家庭问题

    • cause / solve a problem

      引起 / 解决问题

    • I’ve been having problems with my computer system.

      我的电脑系统一直有问题。

  • (考试、教科书中的) 问题：

    • She set her pupils many mathematical problems.

      她给学生们布置了很多数学题。

固定搭配

no problem

[表示乐于相助或事情容易做]没问题：

  • — How about meeting at the nearby restaurant?

    在附近的餐馆见面如何？

  • — No problem.

    没问题。

没关系；不客气：

  • — Thanks for the ride.

    谢谢你载我一程。

  • — No problem.

    不客气。
