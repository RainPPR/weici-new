release

vt 重难点
英音 [rɪ'liːs]
美音 [rɪ'liːs]
词频 4口语 2书面 2阅读 3

英文释义

  • ★ 放开；松开 (to stop holding sth (let go)：)

    • Don’t release the rope.

      别松开绳子。

  • ★ 排放；释放 (to let sb / sth come out of a place where they have been kept or trapped：)

    • release a prisoner

      释放囚犯

    • The factory keeps releasing smoke, making the air dirty.

      这座工厂不停地排放烟雾，污染了空气。

  • ★ 首映 (电影)；发行 (CD、书等)；发布 (新闻等) :

    • release a movie / new version

      发行影片/发布新版本
